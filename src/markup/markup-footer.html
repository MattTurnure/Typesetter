Footer

There is not much visual information in the footer besides a copyright, which should be dynamic. However, all of the JavaScript gets loaded here, so definitely include this in every project.

Gulp will concatentate and minify all of your JavaScript and compile it into one script. This improves performance by making fewer HTTP requests. Enabling gzip and applying expires headers would be a bonus as well. Configuration management can be set in gulpfile.js.

Implementation


        <footer class="footer-main">
            <div class="group">
                <p class="copyright">© 2015 Pattern Library</p>
            </div>
        </footer>
        <script src="/assets/js/main.js"></script>
    </body>
</html>
