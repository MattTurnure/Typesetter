Header

The header should be used as an include with a few dynamic meta parameters that should be passed in:

  • title
  • description
  • bodyClass
  • appName

Implementation


<!DOCTYPE html>
<html class="no-js">
    <head>
        <meta charset="utf-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">
        <title>@@title | Pattern Library</title>
        <meta name="description" content="@@description">
        <meta name="viewport" content="width=device-width, initial-scale=1">

        <link rel="stylesheet" href="//fonts.googleapis.com/css?family=Open+Sans:400italic,300,400,700">
        <link rel="stylesheet" href="/assets/styles/main.css">
    </head>
    <body class="@@bodyClass">
        <header class="header-main">
            <div class="group">
                <h1 class="brand">
                    <a href="/">
                        <img src="/assets/svg/logo.svg" alt="Brand Name">
                        <span class="sub-brand">@@appName</span>
                    </a>
                </h1>
                <a href="#nav-main" class="menu" id="menu">Menu</a>
            </div>
        </header>
